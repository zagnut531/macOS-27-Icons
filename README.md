# macOS 27 Icons

This includes all macOS 27 icons exported from my MacBook Air as PNGs, in original 1024x1024 resolution. I even included CoreServices and other apps. 19 icons have changed (majorly) as of beta 3, so I documented those changes, where other resouces don't.
